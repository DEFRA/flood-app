Rate limiting

We apply rate limiting to Check for flooding to ensure a resilient service.

Per application rates

You can make up to 500 requests within a 1 minute period. If you exceed this limit, you’ll receive a 429 Too Many Requests HTTP status code for each request made in the remainder of the 1 minute window. Your rate limit will reset to its maximum value of 500 requests at the end of the period.
